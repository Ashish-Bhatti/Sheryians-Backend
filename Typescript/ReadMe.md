# Node.js + TypeScript Setup Guide

This guide walks through setting up a Node.js project with TypeScript from scratch.

---

## 1. Create a New Project

Create a new directory and initialize npm:

```bash
mkdir my-app
cd my-app
npm init -y
```

This creates a `package.json` file.

---

## 2. Install Development Dependencies

Install TypeScript and required development tools:

```bash
npm install -D typescript ts-node nodemon @types/node
```

### Package Explanation

| Package | Purpose |
|----------|----------|
| typescript | TypeScript compiler |
| ts-node | Run TypeScript files directly |
| nodemon | Automatically restart server on file changes |
| @types/node | Type definitions for Node.js |

---

## 3. Initialize TypeScript

Generate a TypeScript configuration file:

```bash
npx tsc --init
```

This creates a `tsconfig.json` file.

---

## 4. Configure TypeScript

Replace the contents of `tsconfig.json` with:

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "rootDir": "./src",
    "outDir": "./dist",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  },
  "include": ["src"]
}
```

### Configuration Explanation

| Option | Description |
|----------|----------|
| target | JavaScript version generated by TypeScript |
| module | Module system used by Node.js |
| moduleResolution | How imports are resolved |
| rootDir | Source code directory |
| outDir | Compiled JavaScript output directory |
| strict | Enables strict type checking |
| esModuleInterop | Allows CommonJS package imports |
| skipLibCheck | Speeds up compilation |

---

## 5. Create Project Structure

Create the following folders and files:

```text
my-app/
│
├── src/
│   └── index.ts
│
├── dist/
├── package.json
├── tsconfig.json
└── nodemon.json
```

---

## 6. Create Your First TypeScript File

Create `src/index.ts`:

```ts
function greet(name: string): string {
  return `Hello ${name}`;
}

console.log(greet("Ashish"));
```

---

## 7. Configure Nodemon

Create `nodemon.json`:

```json
{
  "watch": ["src"],
  "ext": "ts",
  "exec": "ts-node src/index.ts"
}
```

### What This Does

- Watches the `src` folder.
- Restarts automatically when `.ts` files change.
- Executes the project using `ts-node`.

---

## 8. Add NPM Scripts

Update the `scripts` section inside `package.json`:

```json
{
  "scripts": {
    "dev": "nodemon",
    "build": "tsc",
    "start": "node dist/index.js"
  }
}
```

### Script Explanation

| Command | Purpose |
|----------|----------|
| npm run dev | Development mode with auto reload |
| npm run build | Compile TypeScript to JavaScript |
| npm start | Run compiled JavaScript |

---

## 9. Run the Project

### Development Mode

```bash
npm run dev
```

Output:

```bash
Hello Ashish
```

---

### Production Build

Compile TypeScript:

```bash
npm run build
```

This creates the `dist` folder containing compiled JavaScript.

Run the compiled application:

```bash
npm start
```

---

# Express + TypeScript Setup

If you're building a backend API, install Express:

```bash
npm install express
npm install -D @types/express
```

---

## Express Example

Create `src/index.ts`:

```ts
import express from "express";

const app = express();

app.get("/", (req, res) => {
  res.send("Hello TypeScript + Express");
});

app.listen(3000, () => {
  console.log("Server running on port 3000");
});
```

Run:

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```

You should see:

```text
Hello TypeScript + Express
```
